# Library Booking

Простой проект на TypeScript + Vite + React для управления бронированием и выдачей книг.

## Быстрый старт

```bash
npm install
npm run dev
```

## Что внутри

- `src/App.tsx` — интерфейс для библиотекаря и читателя
- `src/main.tsx` — точка входа
- `vite.config.ts` — конфигурация Vite
